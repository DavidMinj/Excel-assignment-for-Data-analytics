1. What does the dollar($) sign do?
A dollar sign means a currency format, but it has another common use: indicating absolute cell references in formulas.

2. How to Change the Reference from Relative to Absolute (or Mixed)?
By default, Cell reference is relative. we need to add the dollar sign before the column notation and the row number.
Example to do this operation : A1  is a relative cell reference, and it would become absolute when you make it $A$1.
Or you can use a fast way select  the cell reference then pressing F2, it will change it.

3. Explain the order of operations in excel?
The order of operations is a rule that tells the correct sequence of steps for evaluating a math expression in Excel. We can remember the order using PEMDAS: Parentheses, Exponents, Multiplication and Division (from left to right), Addition and Subtraction (from left to right).

5. When would you use the subtotal function?
The SUBTOTAL function is used when you display a Total row in an Excel Table. Excel inserts the SUBTOTAL function automatically, and you can use a drop-down menu to switch behavior and show max, min, average, etc.
